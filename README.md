# Data-Analysis-Survey-Project-Power-BI
This project analyzes survey responses from over 630 data professionals to uncover insights into salaries, demographics, programming language preferences, job satisfaction, and career trends within the data industry.
Using Power BI and Power Query, raw survey data was cleaned, transformed, and visualized through an interactive dashboard that enables users to explore trends across job roles, countries, genders, and experience levels.
# Key Insights and Features

• Analyzed survey responses from 630+ data professionals.

• Identified salary differences across job roles, countries, and genders.

• Examined popularity of programming languages among data professionals.

• Explored work-life balance and salary satisfaction ratings.

• Visualized respondent demographics including age and gender distribution.

• Investigated perceived difficulty of breaking into the data industry.

• Enabled interactive filtering for deeper exploration of survey results.
